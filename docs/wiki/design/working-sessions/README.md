# Werksessies

Deze map bevat outcomes van werksessies tussen Khidr en Claude waar concrete design-beslissingen worden gemaakt die niet research-shaped zijn — pixel-level keuzes, copy-decisies, specifieke component-instellingen.

## Conventie

Eén bestand per sessie:

```
YYYY-MM-DD-onderwerp.md
```

Voorbeelden:
- `2026-05-03-empty-states-copy.md`
- `2026-05-10-library-card-spacing.md`
- `2026-05-17-welcome-page-flow.md`

## Inhoud per sessie-document

1. **Datum + onderwerp**
2. **Vraag/probleem** waar de sessie mee begon
3. **Beslissingen** (concreet, geen onderbouwing tenzij gevraagd)
4. **Open items** die uit de sessie kwamen
5. **Volgende stap** (Claude Code prompt? volgende sessie?)

## Wanneer een sessie geen wiki-document oplevert

Niet elke sessie hoeft hier vast te leggen — alleen wanneer er substantiële beslissingen zijn die later relevant zijn. Quick fixes of micro-tweaks gaan direct in code en worden via git history achterhaalbaar.
