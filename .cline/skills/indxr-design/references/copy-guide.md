## Key UI Strings

### Credits
- Navbar pill: "🪙 {n} credits"
- Pre-extraction: "This will cost {n} credit(s)"
- Insufficient: "Not enough credits — top up from €4.99"
- Post-deduction: "Done. {n} credit(s) used."

### Extraction states
- Idle: "Paste a YouTube URL or playlist link"
- Loading: "Extracting transcript…"
- Whisper loading: "Transcribing with Whisper AI — this takes 1–3 minutes"
- Success: "Transcript saved to your library"
- No speech: "No speech detected in this video"
- Error (general): "Extraction failed. Please try again."
- Error (blocked): "YouTube blocked access to this video. Try uploading the audio directly."

### Library
- Empty state: "No transcripts yet. Paste a YouTube URL above to get started."
- Search placeholder: "Search your transcripts…"
- Collection label: "Add to collection"

### Duplicate detection
- Prompt: "You've already extracted this video. Extract again anyway?"
- CTA confirm: "Extract again"
- CTA cancel: "View in library"

## Tone of Voice

- Direct. No fluff, no buzzwords.
- Confident but not arrogant — "You never hit a dead end" not "Best-in-class AI transcription™"
- Slightly technical where the user is technical (JSON, SRT, Ctrl+F)
- Human where the user is human ("Actually remember what you watched")
- Never: "revolutionary", "cutting-edge", "seamless", "robust", "leverage"
